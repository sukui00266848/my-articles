# 发布前检查清单（Publish Checklist）

请在发布前逐项确认并填写责任人/时间：

## 内容与合规
- [ ] frontmatter: title/digest/author/cover/series 完整（责任人：__，时间：__）
- [ ] 句长检查通过（脚本结果/人工抽样）（责任人：__）
- [ ] 所有医学/处方/剂量相关段落已提交专家复核并有书面意见（责任人/审核人/时间）
- [ ] 所有引用有至少一个权威来源并填写引用元数据（责任人：__）

## 图片与媒体
- [ ] 所有图片存在对应 .meta.yml 并补全必填字段（alt/photographer/source/license）（责任人：__）
- [ ] 封面尺寸/格式合规（宽度 ≥1200px，WebP 优先，文件大小 < 300KB）（责任人：__）

## 可读性与样式
- [ ] 段落、标题、列表格式符合编辑风格指南（责任人：__）
- [ ] 标题/摘要通过 A/B 初审（如适用）（责任人：__）
- [ ] 图表/表格含 caption 与数据来源（责任人：__）

## 技术与发布
- [ ] 构建器成功生成内联 HTML（无 class/variables 丢失）（责任人：__）
- [ ] contrast-check 脚本通过（text/bg ≥ 4.5:1）（责任人：__）
- [ ] 所有外链/资源可访问（无 404）（责任人：__）

## 发布确认
- [ ] 最终稿件经主编或运营终审（签名/时间）
- [ ] 上线时间/推送渠道确认（运营/增长）
- [ ] 预案：若有用户投诉/版权问题的初步应对计划已就绪（责任人/联系方式）
