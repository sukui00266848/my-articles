# 引用与来源规范（Citation Guidelines）

目的
- 保障文章中事实、古籍与学术引述可溯源、可查证；减少误引与捏造。适用于所有稿件，医学/健康类稿件必须严格遵守并通过二次核验。

一、允许与优先引用源（优先级由高到低）
1. 权威期刊与数据库（PubMed / CNKI / 期刊出版社） — 学术研究事实优先引用此类。
2. 教材与专著（有版本/出版社/ISBN）— 引用需标注页码/章节。
3. 古籍原典（国学馆、中华书局等标注卷/篇/条） — 必须标注卷/章/条/页或典籍子目。
4. 学术综述与专家访谈（注明受访者、职称、访谈日期）
5. 可靠新闻/政府/行业报告（用于事件/政策类）
6. 个人博客/微信公众号等—— 仅作辅助，不作为权威事实来源；引用时需二次核验权威来源。

二、引用格式（必须包含最小元数据）
- 书籍/专著：
  - 格式：作者. 书名. 版次（若有）. 出版地: 出版社, 年份. 页码（若引用）
  - 示例：张三. 中医基础理论（第二版）. 北京: 中医古籍出版社, 2018. 页 123-125
- 期刊文章：
  - 格式：作者. 文章题目. 期刊名. 年;卷(期):起止页. DOI（若有）
  - 示例：李四, 王五. 中医对失眠的辨治研究. 中医药学报. 2020;12(3):45-52. doi:10.xxx/xxx
- 古籍/典籍：
  - 格式：书名. 卷/篇/条（若有）. 译本/校注（若用）. 出版社, 年份（如使用现代影印/译注则注明版本）
  - 示例：《素问》——《素问·六节藏精大论》. 《黄帝内经》校注版, 中华书局, 2005.
- 网页/报告：
  - 格式：作者/机构. 标题. 网站名. 发布日期（或访问日期）. URL
  - 示例：国家卫生健康委员会. 关于XXX的通知. 2022-07-01. https://...

三、引用核验流程（作者提交前的自查）
- 基本核验（作者负责）：
  1. 为每条声明找出 1 个以上权威来源（期刊/教材/古籍）或明确标注为"经验描述/个案/观点"。
  2. 原文逐字比对引用句子，确保无断章取义。
  3. 在稿件的引用处用方括号或脚注标注最小元数据（作者+年份+页码/卷）。
- 深度核验（编辑/研究员在必要时）：
  1. 医学/药事/剂量类内容必须经药师或医务审核并在稿件末附审阅记录（姓名/职称/审核日期）。
  2. 对古籍引用，需上传该古籍的权威版本信息（出版社/卷页/译注），若为二手引文需标注"转引自 X"。

四、引用元数据模板（每条引用至少填写）
- id: AUTOGEN（由系统生成）
- type: book|article|classical|web|report
- author:
- title:
- publisher:
- year:
- page_or_section:
- url:
- notes:（如"转引自XXX"）

五、常见误区（示例）
- 误：把"古籍中的表述"直接当作现代临床建议。
- 纠正：古籍引文应标注出处并注明"为古代文献记载/历史资料"，不可直接用于现代用药建议。

六、最小自动化建议（可立即实现）
- 在构建时自动检测方括号/脚注是否存在来源字段为空；若为空则 PR 阻断并提示作者补全。
- 对于"古籍"关键词（如《素问》《本草》）自动提示上传权威版本信息或标注"二次核验必需"。
